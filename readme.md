# Введение в контроль версий(лекции). Урок 1

## Команды git
1. **git** --version
2. **git** **init**
3. **git** **status**
4. **git** **add** <имя_файла(папки)>
5. **git** **commit** -m "Комментарий комита на английском"
6. **git** **log**
7. **git** **checkout** <минимум 4 символа из номера комита>
8. **git** **chechout** master
9. **git** **diff**
